ThinkPHP 框架开发入门与实战

作者：老六
日期：2025-09-11

什么是 ThinkPHP？

ThinkPHP 是国内流行的 PHP 开发框架，具有高效、简洁、易扩展等特点，适合快速开发各类 Web 应用。

1. 快速安装

推荐使用 Composer 安装：

composer create-project topthink/think tpapp

2. 目录结构简介

tpapp/
├─ app/         # 应用目录
├─ public/      # WEB 入口目录
├─ config/      # 配置文件目录
├─ route/       # 路由定义目录
├─ vendor/      # 第三方库

3. 创建控制器

在 app/controller 目录下新建 Index.php：

<?php
namespace app\controller;

class Index
{
    public function index()
    {
        return '欢迎使用 ThinkPHP！';
    }
}

4. 路由配置

在 route/app.php 添加路由：

use think\facade\Route;

Route::get('/', 'Index/index');

5. 数据库操作示例

ThinkPHP 内置 ORM，操作数据库非常方便：

use think\facade\Db;

// 查询
$list = Db::table('user')->where('status', 1)->select();

// 新增
Db::table('user')->insert(['name' => '老六', 'email' => 'me@example.com']);

6. 返回 JSON 数据

public function api()
{
    return json(['status' => 'success', 'msg' => '接口调用成功']);
}

7. 常见问题

  • URL 404：检查入口文件和路由配置是否正确。
  • 数据库连接失败：确认 config/database.php 配置项。

总结

ThinkPHP 框架适合快速开发中小型项目，文档完善，社区活跃。建议新手多参考官方文档和示例代码，逐步深入学习。
