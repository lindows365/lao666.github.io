ThinkPHP 框架开发入门与实战

作者：老六
日期：2025-09-11

什么是 ThinkPHP？

ThinkPHP 是国内流行的 PHP 开发框架，具有高效、简洁、易扩展等特点，适合快速开发各类 Web 应用。

1. 快速安装

推荐使用 Composer 安装：

composer create-project topthink/think tpapp

2. 目录结构简介

tpapp/
├─ app/         # 应用目录
├─ public/      # WEB 入口目录
├─ config/      # 配置文件目录
├─ route/       # 路由定义目录
├─ vendor/      # 第三方库

3. 创建控制器

在 app/controller 目录下新建 Index.php：

<?php
namespace app\controller;

class Index
{
    public function index()
    {
        return '欢迎使用 ThinkPHP！';
    }
}

4. 路由配置

在 route/app.php 添加路由：

use think\facade\Route;

Route::get('/', 'Index/index');

如何用 Next.js 构建个人博客

作者：老六
日期：2025-08-20

前言

随着前端技术的发展，越来越多的开发者选择使用 Next.js 来搭建个人博客。本文将分享使用 Next.js 快速搭建博客的完整流程和一些实用经验。

1. 项目初始化

首先，使用官方脚手架创建项目：

npx create-next-app my-blog

2. 页面路由与布局

Next.js 支持文件式路由，只需在 pages 或 app 目录下添加对应的文件即可自动生成页面。推荐使用 app 目录实现更灵活的布局。

3. 数据管理

可以使用 getStaticProps 或 getServerSideProps 获取数据，也可以集成 CMS 或数据库。对于个人博客，建议将文章内容以 Markdown 文件存储，便于管理和编辑。

4. 部署上线

推荐使用 Vercel 平台部署 Next.js 项目，只需一键即可自动构建和上线。

5. 常见问题与解决方案

  • 样式丢失：检查 CSS 文件是否正确引入。
  • 路由404：确认页面文件命名和路径是否正确。

总结

Next.js 提供了强大的功能和极佳的开发体验，非常适合个人博客和内容网站的搭建。希望本文能帮助你快速上手并顺利上线自己的博客！
